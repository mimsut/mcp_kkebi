# 기여 가이드 🤝

MCP Kkebi에 기여해주셔서 감사합니다!

## 🚀 빠른 시작

```bash
# 1. Fork & Clone
git clone https://github.com/mimsut/mcp_kkebi.git
cd mcp_kkebi

# 2. 설치
npm install

# 3. 개발 모드 실행
npm run dev

# 4. 빌드 테스트
npm run build
```

## 📝 기여 방법

1. **Issue 생성**: 버그나 기능 제안
2. **Fork**: 리포지토리 포크
3. **Branch**: 새 브랜치 생성 (`feature/amazing-feature`)
4. **Commit**: 명확한 커밋 메시지
5. **Push**: 브랜치에 푸시
6. **Pull Request**: PR 생성

## 🎯 코딩 스타일

- TypeScript 사용
- ESLint 규칙 준수
- 명확한 변수명 사용
- 주석으로 복잡한 로직 설명

## 🐛 버그 리포트

버그를 발견하셨나요? Issue에 다음 정보를 포함해주세요:

- 문제 설명
- 재현 방법
- 예상 동작
- 실제 동작
- 환경 정보 (OS, Node.js 버전 등)

## ✨ 기능 제안

새로운 기능 아이디어가 있으신가요? Issue에 다음을 포함해주세요:

- 기능 설명
- 사용 사례
- 가능하다면 구현 아이디어
